# **Gatinhos Lovers – Plataforma de Adoção de Gatos**

🔗 **Acesse o projeto:** [Gatinhos Lovers](https://liviamoalmeida.github.io/Gatinhos-Lovers-Web-App/)

O **Gatinhos Lovers** é uma aplicação web desenvolvida para simular o funcionamento de uma ONG de adoção de gatos, com foco em navegação intuitiva, organização de dados e experiência do usuário.

O projeto foi estruturado para integrar interface responsiva com manipulação dinâmica de dados, priorizando usabilidade e clareza na apresentação das informações.


## 🐾 **Funcionalidades**

* Listagem dinâmica de gatos disponíveis para adoção
* Sistema de busca por nome e características
* Páginas individuais com informações detalhadas de cada animal
* Exibição de dados como descrição, temperamento e pelagem
* Redirecionamento direto para contato via WhatsApp


## ⚙️ **Implementação Técnica**

* Manipulação de dados em **JSON** para renderização dinâmica de conteúdo
* Uso de **JavaScript** para busca, filtragem e navegação entre páginas
* Estruturação com **HTML5 semântico**
* Estilização com **CSS3**, incluindo responsividade e uso de SVG
* Organização modular de componentes (cards, páginas e layout)


## 🎯 **Diferenciais**

* Interface pensada com foco em **UX/UI minimalista e acessível**
* Layout responsivo adaptado para desktop e mobile
* Estrutura de código voltada para reutilização e manutenção simples
* Integração funcional entre dados e interface


## 🧩 **Tecnologias Utilizadas**

* HTML5
* CSS3
* JavaScript


## 🚀 **Objetivo do Projeto**

Aplicar conceitos de desenvolvimento front-end, manipulação de dados e construção de interfaces funcionais, evoluindo de um exercício inicial para uma aplicação web completa.
